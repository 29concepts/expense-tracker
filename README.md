# Expense Tacker

Expense tracking app built with React Native


### Basic fuctionality

- [] Add/edit/delete expenses
    - [] Date
    - [] Amount
    - [] Description
    - [] Category
    - [] Additional notes
- [] Add income IN (how much money did you make/receive that month)
- [] Expense breakdown by month
- [] Ovrall statistics
- [] Save common options
    - [] payment methods
    - [] places
    - [] categories
